# This is a personal chess project 

The goal of this project that I just started is:

A chess game where the chessboard wouldn't necessary be a 8x8 chessboard.
It could be designed in any shape. And this in order to maybe make chess puzzles, chess 1v1 (multiplayer or agains AI) or other gamemodes yet to define.

A game where chess pieces would behave normally, BUT, there could be more chess pieces with their unique rules. Maybe they could even have abilities ? (who knows x))
